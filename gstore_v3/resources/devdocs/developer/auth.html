Authentication/Authorization

  • GET /apilogin Login to user account with username and password

    Account login

    Note:This route is for use in a scripting environment only.

    Request Parameters

    userid: VW system user id

    password: User password (unencrypted)

    Route Parameters

    None

    Query Parameters

    None

    Examples:
    params = {'userid':'jim_smith','password':'Pass1!_4'}

    Response

    The request returns a text message and cookie in headers -- 'Logged in as userid'.

  • GET /logout Logout of account

    User logout of VW watershed authentication

    Note: This API call is intended to be used in a scripted environment

    Request Parameters

    None

    Route Parameters

    None

    Query Parameters

    None

    Response

    The request returns a text message and disables cookies -- 'Logged out'

  • POST /apicreateuser Create a new user (must be logged in as authenticated user)

    Create new user for VW with authentication (must be logged in as authenticated user)

    Note: This API call is intended to be used in a scripted environment

    Route Parameters

    None.

    Request Parameters

    userid: VW system user id

    firstname: First name of user

    lastname: Last name of user

    email: Email address of user

    password: Password of user (unencrypted)

    address1: First line of address of user

    address2: Second line of address of user

    city: City of user's address

    state: State of user address

    zipcode: Zip code of address of user

    tel_voice: User's voice telephone

    tel_fax: User's fax telephone

    country: User's address country

    Examples:
    params = {'userid':'tsmith','firstname':'Ted','lastname':'Smith','email':'tsmith@gmail.com','password':'pass1_2!T','address1':'12 Rose St.','address2':'University of Idaho','city':'Boise','state':'ID','zipcode':'78543-0001','tel_voice':'202-277-3622','tel_fax':'202-277-3614','country':'USA'}
    Query Parameters

    None

    Response

    The request returns a text string -- "User created:userid"
